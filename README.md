# PixelMotion

웹 기반 도트화 + 픽셀 보정 + 멀티프레임 시퀀스 도구. 좌우 듀얼 슬롯 편집과 floating paste 지원.

## 주요 기능

### 도트화 파이프라인
- **격자 사이즈**: 48 / 64 / 128 / 256 (짧은 변 도트 수, 긴 변은 비례)
- **세분화**: 1×1 ~ 4×4 (큰 격자 한 셀을 sub×sub 도트로 분할)
- **다운샘플 모드**: 평균 / 우세(dominant) / 디테일(채도 가중)
- **색 양자화**: 8 / 16 / 32 / 64색 (Median Cut)
- **채도 부스트**: 기본 / +30% / +60% / +100% (luminance preserving)
- **외곽선**: 끔 / 도트 단위 / 큰 격자 단위
- **적용 범위**: 현재 프레임만 / 모든 프레임 (8방향 일관성 처리)
- **격자 / 세분화 라인** 표시 토글, 격자 자동 미리보기

### 멀티프레임 + 타임라인
- 이미지 여러 장 multi-select 또는 드래그앤드롭으로 한 번에 프레임 추가
- 타임라인 썸네일 + 프레임 추가/삭제/이전/다음
- 재생 ▶ / 정지 ⏸ + FPS 조절
- **어니언 스킨** — 이전/다음 프레임 반투명 겹침
- 통합 저장: 단일 PNG / 모든 프레임 ZIP / 스프라이트 시트 / GIF 애니메이션

### 듀얼 편집 슬롯 (좌우 분할)
- 헤더의 "참조" 드롭다운에서 프레임 선택 → 좌우 50/50 분할
- 양쪽 모두 편집 가능 (브러시/지우개/영역 선택 등 모든 도구)
- **마우스 hover로 활성 슬롯 자동 전환** — 녹색 테두리로 강조
- 두 슬롯이 같은 프레임 가리키면 미러 (어느 쪽에서 변경해도 양쪽 즉시 반영)
- 슬롯 간 클립보드 자동

### 영역 선택 + 색 보정
- **사각형 영역** — 드래그로 직사각형 선택
- **올가미** — 자유 드래그, 떼면 자동 닫힘
- **빈 격자 색 보정**
  - 평균: 영역 안 평균색으로 빈 도트 채움
  - 그라데이션: BFS 거리 기반 시드→평균 점진 수렴

### 픽셀 편집 도구
- **스포이드** — 도트 클릭 → 활성 색 추출
- **브러시** — 활성 색으로 칠하기 (큰 격자 한 셀 또는 1×1)
- **지우개** — 도트를 투명하게

### 복사 / 붙여넣기 / 잘라내기
- **Ctrl+C** — 영역 안 픽셀 복사
- **Ctrl+X** — 잘라내기 (복사 + 영역 안 픽셀 지움)
- **Ctrl+V** — **floating paste**: 마우스 위치에 미리보기로 따라옴 → 클릭으로 확정, ESC로 취소
- 다른 슬롯 / 다른 프레임으로 마우스 이동 후 paste 가능

### 트레이싱 모드 (기름 종이)
- 원본을 가이드로 깔고 트레이스 레이어에 따로 그리기
- 가이드 투명도 슬라이더 (0~100%)
- 저장 시 출력 모드 선택: 합친 결과 / 원본만 / 트레이스만

### 색 / 팔레트
- **활성 색 박스** — 클릭으로 HSV 인라인 픽커 (Hue 슬라이더 + SV 정사각 + HEX 입력)
- **팔레트** — 도트화 결과의 사용 색을 빈도순 자동 추출, 또는 PICO-8 26색 기본 팔레트
- 팔레트 클릭으로 활성 색 변경

### 화면 조작
- **휠 줌** (커서 위치 기준) / **+/-** 단축키 / **0** = FIT / **1** = 100%
- **Space + 드래그** 또는 **Ctrl+Shift + 드래그** — 임시 팬
- **마우스 미들 버튼 드래그** — 팬

### 변환
- **좌우 반전** — 8방향 작업 시 한 방향 결과를 반전해서 반대 방향 자동 생성

### 배경 제거 (rembg)
- **기본** — u2net 모델 (~170MB 첫 다운로드)
- **정밀** — isnet-general-use + alpha matting (~170MB 추가)
- 자동 배경 제거 한계가 있어 외부 툴(Photopea 등)에서 누끼 후 import 권장

### 안전망 + UX
- **Ctrl+Z 되돌리기** — 배경 제거 / 도트화 / 픽셀 편집 / 영역 보정 / floating paste 모두
- **반응형** — 좁은 화면 자동 패널 접기 (1280px / 960px 기준)
- **토스트 알림** — 처리 중 spinner / 성공 ✓ / 실패 ✕
- **빈 상태 안내** — 이미지 없을 때 캔버스 중앙 친절한 안내

## 단축키

| 키 | 동작 |
|----|------|
| `Ctrl+S` | 저장 메뉴 열기 |
| `Ctrl+Z` | 되돌리기 |
| `Ctrl+C` / `Ctrl+V` / `Ctrl+X` | 복사 / floating paste / 잘라내기 |
| `V` / `M` / `L` | 선택 없음 / 사각형 / 올가미 |
| `I` / `B` / `E` | 스포이드 / 브러시 / 지우개 |
| `+` / `-` | 줌 인/아웃 |
| `0` / `1` | 화면 맞춤 / 100% |
| `Space` + 드래그 | 임시 팬 |
| `Ctrl+Shift` + 드래그 | 임시 팬 |
| `ESC` | 모달 닫기 / floating 취소 / 영역 선택 해제 |
| `F1` | 단축키 안내 모달 |

드래그앤드롭으로 이미지 파일을 캔버스에 끌어놓으면 모두 프레임으로 자동 추가.

## 실행

### 의존성
```bash
pip install pillow rembg onnxruntime numpy
```

### 서버 시작
```bash
python server.py
```
브라우저가 자동으로 <http://127.0.0.1:8765/> 로 열립니다.

### 데스크톱 클라이언트 (선택)
PyInstaller로 단일 exe 빌드 가능:
```bash
pip install pyinstaller
pyinstaller --onefile --name PixelMotion ^
  --add-data "index.html;." ^
  --add-data "style.css;." ^
  --add-data "script.js;." ^
  server.py
```
`dist/PixelMotion.exe` 더블클릭 → 콘솔 + 브라우저 자동 오픈.

> 결과물 사이즈 ~2.7GB (rembg + onnxruntime 등 모두 포함). 첫 실행 시 압축 해제 시간 ~5초.

## 추천 워크플로우 — 8방향 캐릭터 시퀀스

```
1. reference 이미지 4~8장 준비 (직접 촬영 / 3D 렌더 / AI 생성)
2. PixelMotion에 드래그앤드롭 → 모두 프레임으로 추가
3. 한 프레임에서 격자/모드/색 조합 결정
4. 적용 범위 = 모든 프레임 → [도트화 실행] → 일관 처리
5. 좌우 반전으로 부족한 방향 채움
6. 듀얼 슬롯 + Ctrl+C/V로 공통 부분(머리 등) 슬롯 간 복붙
7. 트레이싱 모드 + 어니언 스킨으로 자세별 도트 보정
8. [저장] → 스프라이트 시트 PNG 또는 GIF
9. (선택) Aseprite import해서 미세 보정
```

## 한계 (솔직히)

- **자동 도트화로 100% 게임 룩 보장 불가** — 작은 디테일은 격자 사이즈 한계로 손실 불가피. Aseprite 등에서 수작업 보정 필수
- **rembg 배경 제거** — 픽셀 단위 정밀도가 안 됨. 깔끔한 결과는 Photopea 등 외부 툴에서 누끼 후 import
- **AI 합성/스타일 트랜스퍼는 범위 밖** — Stable Diffusion / ComfyUI / PixelLab 등 외부 도구 사용

PixelMotion은 *수작업 비율을 줄이는 자동화 도구*이지 픽셀 에디터 대체품이 아닙니다.

## 자동 테스트

[Playwright](https://playwright.dev/python/) 기반 시나리오 — 서버가 실행 중이어야 함:
```bash
pip install playwright
python -m playwright install chromium
python tests/make_test_image.py   # 입력 이미지 생성
python tests/test_basic.py        # 기본 워크플로우 검증
python tests/test_dual.py         # 듀얼 슬롯 검증
```

스크린샷은 `tests/screenshots/`에 저장 (gitignore).
