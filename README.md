# PixelMotion

웹 기반 도트 아트 + 멀티프레임 애니메이션 + 픽셀 보정 도구.
이미지를 도트로 변환하거나, 처음부터 캔버스에 도트로 그리고, 프레임 시퀀스로 애니메이션을 만들고, 다양한 스타일로 후처리까지.

## 빠른 시작

```bash
pip install pillow rembg onnxruntime numpy
python server.py
```
브라우저가 <http://127.0.0.1:8765/> 로 자동 오픈.

또는 PyInstaller `dist/PixelMotion.exe` 더블클릭.

## 핵심 기능

### 도트화 파이프라인
- **격자 자유 입력 4~512** — 8/16/24/32/36/48/64/96/128/256 datalist 추천
- **세분화 1×1 ~ 4×4** — 큰 격자 한 셀을 sub×sub 도트로 분할
- **다운샘플 모드** — 평균 / 우세(dominant) / 디테일(채도 가중)
- **색 양자화** — 8/16/32/64색 (Median Cut)
- **채도 부스트** — 기본/+30%/+60%/+100% (luminance preserving)
- **외곽선** — 끔 / 도트 단위 / 큰 격자 단위
- **적용 범위** — 현재 프레임 / 모든 프레임 일괄

### AI 자동 정리 ⭐
탑바 체크박스 ON 시 AI 생성 이미지(SD/DALL-E/Midjourney)를 드래그앤드롭하면:
1. 권장 격자 자동 선택 (1024 → 64, 512 → 32 등)
2. 격자에 맞게 bilinear 리사이즈 (800×600 → 768×576)
3. 즉시 도트화 실행

→ 한 번에 깔끔한 도트 캔버스. OFF 상태에선 권장 격자만 토스트로 안내.

### 격자 추가 (디테일 단계) ⭐
도트 작품을 보존하며 단계별로 디테일 추가:
```
36×36 러프 → ×4 → 144×144 디테일 → ×2 → 288×288 마감
```
- NEAREST 보존 업스케일 — **그린 도트는 N×N 블록으로 그대로 보존**
- 매 단계마다 더 잘게 그리기 가능
- 사용자 정의 시작 사이즈 자유 (36, 80 등 임의값)

### 셀 변주 + 명암 자동 추천 (셰이딩 보조)
- **셀 변주**: 큰격자 브러시 칠할 때 sub×sub 안 명도 미세 변주 (끔/약/중/강) — 결정론적 노이즈
- **명암 strip**: 활성 색 아래 V 단계 4개 (`──/−/+/++`) — 클릭만으로 어두운/밝은 단계 즉시 활성

### 좌우 대칭 브러시 ⭐
도구 패널 토글. ON 시 한쪽 칠하면 반대쪽 자동 미러. 캐릭터 정면 그릴 때 시간 50% 절약.

### 픽셀 편집 도구
- **브러시** — 활성 색 (큰 격자 / 1×1) — **우클릭 = 임시 지우개** (Aseprite 스타일)
- **지우개** — 도트 투명화
- **스포이드** — 도트 클릭 → 색 추출
- **선택** (사각형 / 올가미) — 라디오 클릭만으로 즉시 활성

### 멀티프레임 + 타임라인
- 이미지 multi-select / 드래그앤드롭 → 모두 프레임 추가
- **+ 빈 프레임** — 현재 사이즈로 빈 캔버스 추가
- **+ 복제 (Ctrl+D)** — 현재 프레임 통째 복제 (트레이스 포함)
- **← / →** — 이전/다음 프레임 빠른 전환
- 재생 모드: **루프 / 원샷 / 핑퐁** (왕복)
- **어니언 스킨** — 끔 / ±1 / ±3 / 모두 + 색 틴트 (이전 빨강 / 다음 파랑)
- 드래그로 프레임 순서 변경

### 레이어 패널 (Photoshop 스타일) ⭐
프레임 = 레이어 (위 = 앞):
- 👁 표시 토글
- 썸네일
- 이름 (더블클릭 편집)
- 불투명도 슬라이더
- 드래그 재정렬

**모드 토글**:
- 🎬 **애니메이션** — 활성 프레임 1장만 (재생 시 순환)
- 🍰 **레이어** — 표시 ON 모든 프레임 동시 겹쳐 보기 (불투명도 적용)

### 미리보기 모달
탑바 [▶ 미리보기] — 별도 창에서 ×1/×2/×4/×8 동시 재생, FPS/모드 독립 조정.

### 영역 선택 + 편집
- **사각형 / 올가미** 영역 선택
- **빈 격자 색 보정** — 평균 / 그라데이션 (BFS 점진 수렴)
- **Delete** — 선택 영역 비우기
- **Ctrl+Shift+I** — 선택 반전

### 복사 / 붙여넣기 / 회전 ⭐
- **Ctrl+C** / **Ctrl+X** / **Ctrl+V** — floating paste (마우스 따라옴)
- **R** — 90° 회전 (NEAREST 픽셀 재정의 — 보간 없음, 도트 룩 보존)
- **Shift+R** — 임의 각도 입력 (예: 37°)
- 캔버스에 각도 오버레이 표시
- 클릭으로 확정 / ESC 취소

### 트레이싱 모드
- 원본을 가이드로 깔고 별도 trace 레이어에 그리기 (어니언 + 멀티프레임 결합 시 모션 작업)
- 가이드 투명도 슬라이더 (0~100%)
- 출력 모드: 합친 결과 / 원본만 / 트레이스만

### 스타일 변환 ⭐
도트 작품을 5가지 스타일로 변환 (라이브 슬라이더 미리보기):
- 🎨 **수채화 (정통)** — Sobel 엣지 진해짐 + 종이 결 + 톤 압축 + 색 번짐
- 🖌 **유화** — 큰 붓터치 + 진한 채도
- ✏ **연필 스케치** — 회색조 + 엣지 강조 + 종이 결
- 🖋 **잉크** — 깔끔 색 채움 + 검은 라인
- 💨 **소프트 블러** — 단순 흐림 (가벼움)

각 스타일마다 슬라이더 4개 동적 구성, [PNG 저장] 또는 [현재 프레임에 적용] (Ctrl+Z 가능).

### 좌우 반전
- 우측 도구 패널 [↔ 좌우 반전] — 현재 프레임만
- **Shift+클릭** — 모든 프레임 일괄 (8방향 작업용)

### 색 / 팔레트
- **활성 색 박스** — 클릭으로 HSV 인라인 픽커 (Hue 슬라이더 + SV 정사각 + HEX 입력)
- **명암 strip** — 활성 색의 V −25%/−12%/+12%/+25% 4단계
- **팔레트** — 도트화 결과 사용 색 빈도순 자동 추출 또는 PICO-8 풍 기본 26색

### 듀얼 편집 슬롯 (좌우 분할)
- 타임라인의 "참조" select에서 프레임 선택 → 좌우 50/50 분할
- 양쪽 모두 편집 가능 (모든 도구 작용)
- 마우스 hover로 활성 슬롯 자동 전환 (녹색 테두리 강조)

### 배경 제거 (rembg)
- **기본** — u2net 모델
- **정밀** — isnet + alpha matting (가장자리 더 깔끔)
- 첫 호출 시 모델 ~170MB 다운로드

### 화면 조작
- **휠 줌** (커서 위치 기준)
- **+/-** 줌 인/아웃 / **0** = FIT / **1** = 100%
- **Space + 드래그** / **Ctrl+Shift + 드래그** / **마우스 미들 버튼** — 팬

### 안전망
- **Ctrl+Z 되돌리기** — 도트화 / 배경 제거 / 픽셀 편집 / 영역 보정 / floating paste / 회전 / 스타일 적용 / 격자 확대 모두 (20단계)
- **반응형** — 좁은 화면 자동 패널 접기 (1280px / 1000px)
- **헬프 아이콘 (?)** — 모든 패널 헤더 호버 시 사용법 툴팁
- **토스트 알림** — 처리 중 spinner / 성공 ✓ / 실패 ✕

## 추천 워크플로우

### 1) AI 이미지 → 깔끔한 도트
```
1. 탑바 [AI 자동 정리] 체크 ON
2. AI 생성 이미지(1024×1024 등) 드래그앤드롭
3. → 자동으로 격자 64 + 도트화 즉시 완료
4. 필요하면 셰이딩 보정 (활성 색 + 명암 strip)
5. 저장 → PNG / 스프라이트 시트 / GIF
```

### 2) 처음부터 캐릭터 그리기 (디테일 단계)
```
1. 탑바 격자 = 32 → 32×32 빈 캔버스
2. 좌우 대칭 브러시 ON → 캐릭터 정면 한쪽만 그리면 미러
3. 명암 strip 클릭 → 어두운/밝은 단계로 셰이딩
4. 우측 [격자 추가] ×2 → 64×64 (디테일 추가)
5. 만족하면 저장
```

### 3) 애니메이션 시퀀스 (걷기/공격 등)
```
1. 격자 32 + 캐릭터 1프레임 그리기
2. Ctrl+D 복제 → 자세 살짝 수정
3. 어니언 스킨 ±1 ON → 직전/다음 프레임 비교하며 정렬
4. Ctrl+D 반복 → 키 프레임 4~8장
5. 트레이싱 레이어로 검 궤적 / VFX 추가
6. 탑바 [▶ 미리보기] → ×1/×2/×4/×8 동시 재생 확인
7. 모드 = 핑퐁/루프 → 자연스러운지 검증
8. 저장 → GIF 또는 스프라이트 시트
```

### 4) 도트 → 수채화/유화 풍 출력
```
1. 도트 작품 완성
2. 우측 [🎨 스타일 변환] → 모달
3. 스타일 select (수채화/유화/연필/잉크/소프트)
4. 슬라이더 조절하며 라이브 미리보기
5. PNG 저장 또는 [현재 프레임에 적용]
```

## 한계 (솔직히)

- **자동 도트화로 100% 게임 룩 보장 불가** — 작은 디테일은 격자 사이즈 한계로 손실. 수작업 보정 필수
- **rembg 배경 제거** — 픽셀 단위 정밀도 안 됨. 깔끔한 결과는 외부 툴(Photopea 등) 권장
- **수채화 변환** — 진짜 수채화 시뮬레이션 아님. "edge bleed + 종이 결 + 톤 압축" 정도 흉내
- **AI 합성/스타일 트랜스퍼는 범위 밖** — Stable Diffusion / ComfyUI / PixelLab 등 외부 도구

PixelMotion은 *수작업 비율을 줄이고 워크플로우를 정렬하는 도구*이지 픽셀 에디터 대체품이 아닙니다.

## 빌드 (선택)

```bash
pip install pyinstaller
pyinstaller PixelMotion.spec
# 또는:
pyinstaller --onefile --name PixelMotion ^
  --add-data "index.html;." --add-data "style.css;." --add-data "script.js;." ^
  server.py
```

`dist/PixelMotion.exe` 더블클릭 → 콘솔 + 브라우저 자동 오픈.

> 결과물 ~2.7GB (rembg + onnxruntime 포함). 첫 실행 시 압축 해제 ~5초.

## 자동 테스트

[Playwright](https://playwright.dev/python/) 시나리오 — 서버 실행 중 필요:
```bash
pip install playwright
python -m playwright install chromium
python tests/make_test_image.py   # 입력 이미지 생성
python tests/test_basic.py        # 기본 워크플로우
```

> 기존 테스트는 UI 개편으로 셀렉터가 깨졌을 수 있음 — 별도 갱신 필요.
